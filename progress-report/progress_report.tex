    \begin{frame}{Project Overview}
        \begin{itemize}
            \item \textbf{Goal}: design and implement a simply-typed lambda calculus with graded modality to enforce secure information flow (noninterference property).
            \item \textbf{Core ideas}: use a graded necessity $\square_\ell T$ on a security semiring $\{\texttt{Pub}\sqsubseteq\texttt{Sec}\}$ to track the security requirement of information.
        \end{itemize}
        The project (implementation, report draft, and Lean 4 formalization) is available at https://github.com/nofe1248/cs315-term-project/.
    \end{frame}
    
    \begin{frame}{Current Progress}
        Completed:
        \begin{itemize}
            \item Lexer and parser are implemented using ANTLR4.
            \item AST generation completed and integrated with parse trees.
            \item Complete specification of the syntax, typing rules, and small-step operational semantics of our type system.
        \end{itemize}
        Ongoing:
        \begin{itemize}
            \item Type checker implementation.
            \item AST-based interpreter.
            \item Lean 4 formalization of the type system and relevant proofs: about halfway finished.
        \end{itemize}
    \end{frame}
    
    \begin{frame}{Next Steps}

            \item Prepare a demo and evaluation: write example programs, finish the project report, etc. (1 week).

    )
